What are React Hooks?

Hooky is a mechanism in React that allows you to work entirely without classes.

Slide 3

Advantages of using React Hooks

• More flexibility in reusing an existing piece of code.

• There is no need to refactor the functional component into a class component when it grows complex.

• No more bindings for methods

• It is simpler to distinguish logic and UI using hooks.

useEffect()

The useEffect hook is useful when you wish to run some functions during the component's lifecycle.


						useEffect(callback[, dependencies]);

callback is the function containing the side-effect logic. Callback is executed right after changes were being pushed to DOM.

Dependencies is an optional array of dependencies. useEffect() executes callback only if the dependencies have changed between renderings.

useContext()

Context provides a way to pass data or state through the component tree without having to pass props down manually through each nested component.


						const value = useContext(SomeContext)

useMemo()

useMemo() is a built-in React hook that accepts 2 arguments — a function compute that computes a result and the dependencies array:
